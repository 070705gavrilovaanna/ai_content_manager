# AI Контент-менеджер

Веб-приложение для создания, хранения и управления контентом с помощью AI. Генерирует статьи, посты, идеи и контент-планы, автоматически сохраняет всё в локальные файлы. Использует OpenRouter API для доступа к LLM.

Блогерам, авторам, smm-специалистам - быстро получать черновики, не копировать их вручную, не терять в истории чатов. Все хранится локально, ищется, публикуется.

**Технологии:** Python, LangChain, FastAPI, OpenRouter API, Jinja2, Markdown

---

## Возможности

- Генерация статей и постов на любые темы
- Генерация идей для контента
- Создание контент-планов на любой период
- Автоматическое сохранение статей и постов в черновики (папка drafted)
- Публикация черновиков (копирование в папку published)
- Поиск по всему сохраненному контенту
- Статистика по созданному контенту
- Веб-интерфейс для работы в браузере
- Консольный режим для тестирования

## Установка и запуск

### 1. Клонирование

```bash
git clone https://github.com/your-username/content_ai_manager.git
cd content_ai_manager
```

### 2. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 3. Настройка API ключа OpenRouter

1. Зарегистрируйся на [openrouter.ai](https://openrouter.ai)
2. В разделе Settings -> Keys создай API-ключ
3. Создай файл `.env` в корне проекта:

```
OPENROUTER_API_KEY=sk-or-v1-твой-ключ
```

### 4. Запуск

**Веб-интерфейс:**

```bash
python main.py
```

Открой браузер: http://localhost:8000

**Консольный режим (для тестирования):**

```bash
python agent.py
```

## Команды

| Команда | Действие |
|---------|----------|
| `идея [тема]` | Генерирует 5 идей для контента |
| `сохранить идеи [тема]` | Генерирует и сохраняет идеи в `ideas.json` |
| `мои идеи [тема]` | Показывает сохранённые идеи по теме |
| `добавить идею [тема] [идея]` | Добавляет идею вручную |
| `темы` | Показывает все темы с идеями |
| `план [тема] [дни]` | Создаёт контент-план |
| `статья [тема]` | Пишет статью и сохраняет в `drafts/` |
| `пост [тема]` | Пишет пост и сохраняет в `drafts/` |
| `сохранить [название] [текст]` | Сохраняет любой текст в `drafts/` |
| `опубликовать [название]` | Копирует черновик в `published/` |
| `поиск [запрос]` | Ищет по всем сохранённым файлам |
| `статистика` | Показывает статистику по контенту |

**Примеры:**

```
идея бизнес на маркетплейсах
статья как начать свой бизнес с нуля
план здоровый образ жизни 7
пост мотивация на каждый день
поиск бизнес
статистика
```

## Как это работает

### Генерация контента

1. Пользователь отправляет команду (например, `статья как начать бизнес`)
2. AI генерирует текст через OpenRouter API
3. Результат автоматически сохраняется в `data/drafts/` как MD-файл
4. Файл отображается в веб-интерфейсе

### Публикация

1. В разделе "Файлы" (`/stats`) отображаются все черновики
2. При нажатии "Опубликовать" файл копируется в `data/published/`
3. Черновик остаётся в папке drafts (не удаляется)

### Поиск

- Поиск работает по обеим папкам (`drafts` и `published`)
- Ищет по ключевым словам внутри текста
- Регистр не важен

### Статистика

Показывает количество черновиков, опубликованных материалов и сохранённых идей.

## Настройка моделей

В файле `config.py` можно указать модель OpenRouter:

```python
OPENROUTER_MODEL = "mistralai/mistral-7b-instruct:free"
```

**Бесплатные модели (на момент написания):**

- `meta-llama/llama-3.1-8b-instruct:free`
- `microsoft/phi-3-mini-128k-instruct:free`

Актуальный список бесплатных моделей смотри на [openrouter.ai/models](https://openrouter.ai/models) (фильтр Free).

## Ошибки и их решение

### RateLimitError (429)

OpenRouter ограничивает бесплатные модели при высокой нагрузке.

**Решение:**
- Подожди 30 секунд и попробуй снова
- Смени модель в `config.py` (например, на `meta-llama/llama-3.1-8b-instruct:free`)
- Зарегистрируйся на OpenRouter и добавь свой API-ключ (бесплатно, увеличивает лимиты)

### TemplateNotFound

Ошибка при открытии страницы `/ideas` или `/stats`.

**Решение:** Проверь, что в папке `templates/` есть файлы `ideas.html` и `stats.html`. Если нет — создай их по шаблонам из репозитория.

### Файлы не сохраняются

**Решение:**
- Проверь, что папка `data/drafts/` существует (создаётся автоматически при первом запуске)
- Проверь права на запись в папку `data/`

## Сравнение: веб-интерфейс vs консоль

| Критерий | Веб-интерфейс | Консоль |
|----------|---------------|---------|
| Удобство | Высокое (кнопки, ссылки) | Среднее (только текст) |
| Отображение Markdown | Да (с форматированием) | Нет (сырой текст) |
| Управление файлами | Да (кнопки Просмотр/Опубликовать) | Только командами |
| Скорость | Медленнее (загрузка страниц) | Быстрее |
